About The Tool

The purpose of this project is to demostrate world-wide social economics. With the data we grabbed and the visualization we gave for the data, we can conclude a lot of neat scenarios. For example, we found the top 5 highest birth/death ratio countries. We also found out that the birth rate in China signically decreased after the 1970's. Lastly, our favorite visualization is the scatter plot of future population. The reason this data visualization is so cool is because the data was found using Microsoft Azure using a Boosted Decision Tree Regression algorithm.


More About The Tool

In this tool we are using D3 (Data-Driven Documents) to show case our data visualizations on social economics.

Even Even More...

Not only did we use D3.js, we also used Three.js. Using Three.js was a great framework because it delivered 3D graphs for our data. Three.js is a library that makes WebGL - 3D in the browser - easy to use. While a simple cube in raw WebGL would turn out hundreds of lines of Javascript and shader code, a Three.js equivalent is only a fraction of that.

Why?

This project was inspired by Wild Hacks 2015 and Big Data. Our goal for this project is to take real data and make it come to life! This helps visualize world-wide social economoics.
